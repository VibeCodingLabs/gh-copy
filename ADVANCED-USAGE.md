# ADVANCED-USAGE.md

## Remote Repository Packing (Coming Soon)
Future updates will allow packing directly from a GitHub URL without cloning the repository locally, mimicking the functionality of `gitingest` and `repomix --remote`.

## Templates (Coming Soon)
Support for Jinja2 templates is planned to allow users to customize the output format (e.g., switching between Markdown blocks and XML tags `<file name="...">`) to suit different LLM preferences.
